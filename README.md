# MVV
- 日本のインターネット企業を中心にMission・Vision・Valueおよびそれに似た概念（PurpopseやPhilosophyなど）を個人的にまとめたものです
  - 公開情報をもとに作成しています
  - 順番に深い意図は特にありません
- MVVは企業の北極星だと思っているので、誤りがないように注意を払いながら記載していますが、内容の正確性を必ずしも保証するものではありません
  - 古いバージョンのMVVが記載されている可能性もあります 
  - もし、誤りや変更があった際はPull Requestをいただければ幸いです
- 追加のPull Requestも大歓迎です！
  - 細かいルールは特にありません
